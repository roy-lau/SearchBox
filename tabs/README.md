##  tabs选项卡切换(插件)

### 用法一：

	$(".js-tab").tab().css("backgroundColor","red")

注      

 .js-tab  	// HTML DOM集合
 tab()  	// 触发插件的方法
 tab()....   // 链式调用


### 用法二：

	Tab.init($(".js-tab")) 

注 

  tab.init()   // 初始化插件
  $(".js-tab")  // HTML DOM 集合
